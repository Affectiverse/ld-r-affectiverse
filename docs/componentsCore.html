Components: Core

Dataset Level

Dataset
Collection of resources.
ResourceList
A simple list of resources.
ResourceListPager
Pagination for resource list.
FacetedBrowser
For browsing a collection of resources.

Resource Level

Resource
Collection of properties.

Property Level

IndividualProperty
For handling an individual property of a resource.
PropertyHeader
Handles the property lable and hint.

Object Level

IndividualObject
Applies to each individual object of a property.
AggregateObject
Applies to whole objects of a property.
ObjectIEditor
Dispatcher for individual edit components.
ObjectIViewer
Dispatcher for individual view components.
ObjectAEditor
Dispatcher for aggregate edit components.
ObjectAViewer
Dispatcher for aggregate view components.
ObjectBrowser
Dispatcher for browse components.


Viewer

Individual

BasicIndividualView
Default viewer for an individual object.
BasicLinkedIndividualView
Default viewer for an extended object. Use shortenURI parameter to shorten the URIs or use linkedGraph if the object is linked to another graph than the current graph.
BasicIndividualDetailView
Default viewer for details of an extended object.

Aggregate

BasicAggregateView
Default viewer for aggregate objects.

Editor

Individual

BasicIndividualInput
Default editor for individual objects.
BasicIndividualDetailEdit
Default editor for extended object mode.

Aggregate

BasicAggregateInput
Default editor for aggregate objects.

Browser

CheckboxItem
Default browser for facets.

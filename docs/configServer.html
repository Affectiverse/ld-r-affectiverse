Config: Server

For server config, you have to change the following parameters in configs/general.js:

Parameter Description Example value
serverPort The port number of your application. [4000]
sparqlEndpoint The specification of your SPARQL endpoint. It should contain the graph name on which your data can be found. 'http://dbpedia.org': { host: 'dbpedia.org', port: 80, path: '/sparql' }
dbpediaLookupService Address of your DBpedia lookup service. { host: 'lookup.dbpedia.org' }

Config: General

Parameter Description Example value
appFullTitle The title which appears as page title of your application. ['Linked Data Reactor']
appShortTitle Short title of your application. ['LD-Reactor']
serverPort The port number of your application. [4000]
sparqlEndpoint The specification your SPARQL endpoint. It should contain the graph name on which your data can be found. 'http://dbpedia.org': { host: 'dbpedia.org', port: 80, path: '/sparql' }

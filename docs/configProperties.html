Config: Properties

Properties config are defined per graph. The 'generic' property config will be used if no specific config is defined for a graph. For properties config, you have to change the following parameters in configs/reactor.js:

Parameter Description Example value
readOnly If set to '1', will disabled editing property values. 0
useCategories If set to '1', you can categorize properties in different tabs. It is required to define the list of categories beforehand by using categories. 1
categories List of categories to be used. ['general', 'content', 'structural']
